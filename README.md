# odin-Restaurant_Page


A single-page restaurant website built with **Vanilla JavaScript** and **Webpack**, featuring dynamic tab navigation.



## Features

- Home, Menu, and About pages generated dynamically with JS modules  

- Tab switching without page reload  

## Tech

- JavaScript (ES6 modules), Webpack, HTML, CSS
